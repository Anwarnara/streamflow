# RBAC & Permissions Matrix
**Version:** 3.1.0  
**Last Updated:** 2026-07-17  

## 1. Role Definitions
- **Admin**: Akses penuh ke seluruh fitur.
- **Member**: Akses terbatas pada entitas yang dimiliki.

## 2. Matrix
| Fitur / Endpoint | Admin | Member |
|------------------|-------|--------|
| Live Simulcast   | Yes   | Yes (Own)|
| Fallback Config  | Yes   | Yes (Own)|
| User Management  | Yes   | No     |
